    module DICOMElementsHelper

    Noms et valeurs possibles de quelques éléments de données DICOM qui nous intéressent.

    Constants

    DICOM_ELEMENTS_DICT

    Dictionnaire de tous les éléments DICOM, avec leur nom et type

    DICOM_EXPOSURE_TIME_CSVNAME
    DICOM_EXPOSURE_TIME_TAG

    Durée d’exposition de la tranche (en ms)

    DICOM_FILTER_TYPE_CSVNAME
    DICOM_FILTER_TYPE_TAG

    Filtre appliqué par le scanner aux tranches d’une série d’images;

    1. ex. ‘HEAD FILTER’, ‘BODY FILTER’, etc.

    DICOM_IMAGE_TYPE_CSVNAME
    DICOM_IMAGE_TYPE_SCOUT_MODE_RE

    Valeurs possible de l’élément 0008,0008 (Image Type): expression régulière permettant d’identifier une image « scout »

    DICOM_IMAGE_TYPE_TAG

    Type de l’image; contient trois valeurs séparées par des backslash ‘', telles que ’ORIGINALPRIMARYAXIAL’, ‘ORIGINALPRIMARYLOCALIZER’ ou ‘DERIVEDSECONDARYSCREEN SAVE’

    DICOM_MODALITY_CSVNAME
    DICOM_MODALITY_TAG

    Type de scanner; devrait toujours valoir ‘CT’ dans notre cas

    DICOM_PIXEL_DATA_CSVNAME
    DICOM_PIXEL_DATA_TAG

    “7FE0,0000” (ne possède pas de nom)

    DICOM_PIXEL_GROUP_LEN_CSVNAME
    DICOM_PIXEL_GROUP_LEN_TAG

    Données du bitmap d’une image

    DICOM_SCAN_OPTIONS_CSVNAME
    DICOM_SCAN_OPTIONS_TAG

    Mode d’acquisition d’une série d’images; p. ex. ‘SCOUT MODE’ ou ‘HELICAL SCAN’. Noter que cet élément n’est pas présent sur tous les examens et qu’il est plus sûr de chercher si c’est une image « scout » dans l’élément Image Type (voir DICOM_IMAGE_TYPE_TAG).

    DICOM_SERIES_DESCRIPTION_CSVNAME
    DICOM_SERIES_DESCRIPTION_TAG

    Description d’une série d’images

    DICOM_SLICE_LOCATION_CSVNAME
    DICOM_SLICE_LOCATION_TAG

    Distance physique relative entre la tranche correspondant à une image et la tranche de la première image

    DICOM_SOURCE_FILE_CSVNAME
    DICOM_SOURCE_FILE_TAG

    Pseudo-élément correspondant au nom du fichier source d’une image

    DICOM_XRAY_TUBE_CURRENT_CSVNAME
    DICOM_XRAY_TUBE_CURRENT_TAG

    Intensité du courant pendant l’acquisition

    csv_colname_for( element_tag) click to toggle source

    Compose et retourne le nom de colonne qui doit figurer dans un en-tête CSV pour un élément DICOM désigné par son nom (tag). Si le nom de l’élément ne figure pas dans le dictionnaire du module DICOM, retourne inchangé ce nom d’élément.

    # File lib/dicom_elements_helper.rb, line 19
def self.csv_colname_for( element_tag)
  DICOM_ELEMENTS_DICT.has_key?( element_tag)        ? "#{element_tag} (#{DICOM_ELEMENTS_DICT[ element_tag][ 1]})"        : "#{element_tag}"
end
