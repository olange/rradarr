    html_graph_for( data, title) click to toggle source

    Retourne un document HTML permettant de tracer un histogramme à partir d’un ensemble de coordonnées <Slice location, X-ray tube current>.

    Le jeu de données doit être communiqué sous forme d’un tableau de coordonnées, où chaque coordonnée doit être un hash de la forme { :l => FLOAT, :x => FLOAT }. Par exemple:

    { :l => 66.125, :x => 140.0 }, { :l => 66.750, :x => 145.5 }, … 
    # File lib/graph_template.html.rb, line 16
def html_graph_for( data, title)
  return BARGRAPH_TEMPLATE      % { :graph_data => data.to_json, :graph_title => title }
end
